# Overview

Bu laboratuvarda, bir web uygulamasında bulunan
**SQL Injection (SQLi)** zafiyeti test edilmiştir.

Amaç, kullanıcıdan alınan girdilerin sunucu tarafında
yeterince kontrol edilmemesi durumunda,
veritabanı sorgularının nasıl manipüle edilebildiğini
pratik olarak gözlemlemektir.

Bu çalışma, yetkili bir laboratuvar ortamında,
eğitim ve öğrenme amacıyla gerçekleştirilmiştir.
