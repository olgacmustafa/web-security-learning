- **Overview** ‚Äì Scope, objective, and attack surface  
- **Methodology** ‚Äì Reconnaissance and vulnerability identification approach  
- **Exploitation** ‚Äì Step-by-step exploitation of identified weaknesses  
- **Impact / Results** ‚Äì Security impact and achieved level of access  
- **Mitigation** ‚Äì Recommended defensive and secure coding measures
