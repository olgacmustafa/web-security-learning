# Web Security Learning Repository

This repository contains **educational and authorized web security lab write-ups**.
It is intended to document my learning journey in web application security.

## Topics Covered
- SQL Injection
- Brute Force Attacks
- File Upload Vulnerabilities
- Authentication Bypass
- Remote Code Execution (RCE)

## Structure
Each lab includes:
- Overview
- Exploitation methodology
- Impact analysis
- Mitigation techniques
- Lessons learned

## Disclaimer
All labs were completed in **legal training environments**.
No real-world systems were targeted.
